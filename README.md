# My Personal Website

This website is used to list some of my tech blog, the introduction of myself and some other thigns

## Git Page Link

Visit: [https://mwfj.github.io](https://mwfj.github.io)

## Web Features Design

- **Responsive Design** - Optimized for desktop, tablet, and mobile devices
- **Dark/Light Mode** - Toggle between themes with preference saved to localStorage
- **Smooth Animations** - Typing effect, fade-in on scroll, animated skill bars
- **Side Navigation** - Icon-based navigation that expands on hover
- **Print-Friendly Resume** - Dedicated resume page with print styles

## Pages

| Page | Description |
|------|-------------|
| `index.html` | Main landing page with hero, about, skills, portfolio, and blog sections |
| `pages/resume.html` | Full resume with work experience, education, and skills |
| `pages/blogs.html` | Blog article listing with category filters |
| `pages/blog-post.html` | Individual blog post template |

## Tech Stack

- **HTML5** - Semantic markup
- **Tailwind CSS** - Utility-first CSS framework (via CDN)
- **Vanilla JavaScript** - No frameworks, pure JS for interactivity
- **Google Fonts** - Inter, Poppins, Noto Sans SC
- **Material Icons** - Outlined icon set
